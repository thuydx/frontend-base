# frontend-base
This is git repository template for the frontend projects based on the NextJS framework and CoreUI.

## Technology Stack
This project uses:

- Next.js 16.2.9 with App Router
- React 19 with functional components and hooks
- TypeScript in strict mode
- Redux Toolkit with React Redux
- CoreUI React PRO 5.26 and CoreUI PRO 5.26
- Sass/SCSS
- FullCalendar 6.x
- Google Maps integration
- Chart.js integrations via CoreUI wrappers
